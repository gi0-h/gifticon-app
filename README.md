# gifticon-app
모바일 프로그래밍 프로젝트

## 팀구성
이름|역할
:---:|:---:
이승원|메이페이지, 추가, 캘린더등
황기영|데이터베이스, 상세, 알림등

## 어플 소개
- 갤러리에 추가된 기프티콘 이미지를 어플에 추가하여 관리하는 어플
- 어플의 만료일자 설정 및 알림
 - 캘린더에 기프티콘에 대한 간단한 메모
